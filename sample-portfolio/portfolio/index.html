Technetium

JavaScript • Python • Django • D3 • Bitbucket API

A Bitbucket data aggregation tool built using Bitbucket's RESTful API. Features a centralized issue tracker, repository subscription manager, and reports/graphs which enumerate the commit/issue activity of each user in a subscribed repository.

Technetium's frontend is build using Twitter Bootstrap. The centralized issue tracker gathers data through Bitbucket's API endpoints, and is sped up considerably using Python's gevents library for sending asynchronous requests, while being highly responsive using AJAX to load/filter issue data.

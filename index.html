David Leonard

I am a Master's student at the City College of New York studying Computer Science, where I am also a researcher at G.L.A.S.S Lab under Professor Grossberg's lead. Currently I am taking two classes: Formal Language Theory, and Compiler Construction where I will be developing a Pascal Compiler using Python/JavaScript.

Throughout my undergraduate years, I dabbled in many different areas of Computer Science: Cryptography, Web Technologies, Game Development, Machine Learning and Artificial Intellignece. I've programmed using over a dozen programming languages including Lua, Scala, Haskell, Pascal, Python, C++/C, Java, Rust and my favorite: JavaScript.

Recently, I've been doing quite a bit of programming using Rust - a relatively new language built by Mozilla in 2011. I've even contributed to open source projects revolving around Rust, such as rust-by-example, a collection of examples highlighting usage of Rust and all that it provides. The Rust compiler is currently bleeding edge and examples are commonly getting deprecated, but it has kept me on my toes and keeping up with changes to the language and the compiler itself.

My passion lies within Web Technologies - from consuming APIs and building my own, to building full-stack web applications using Node.js, Angular.js, MongoDB and Express.js.

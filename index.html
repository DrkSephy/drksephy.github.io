~/ About Me

Hey, I'm David. I'm a software developer whose passion lies in gaming, simulations and all things JavaScript. I once held a Github streak of 800+ days (inspired by John Resig's Write Code Every Day) between February 19th, 2014 to May 26th, 2016 during which time I ventured into the world of JavaScript and since then haven't looked back. My passions lie in game development, web applications and visualizations, to name a few.

I currently work at Yahoo on the Yahoo Developer Network team where I work on front-end (React.js + Redux), back-end (node.js, express.js) and occasional devops. Over my ~8 month tenure, I've worked on implementing API consoles purely in HTML, CSS and JavaScript, new components for our CMS architecture, a custom search platform supporting contextual search across domains and various internal projects while leveraging the latest JavaScript ecosystem (React, Redux, Immutable, Webpack, ES6).

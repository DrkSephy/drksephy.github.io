Technetium

JavaScript • Python • Django • D3 • Bitbucket API

A Bitbucket data aggregation tool built using Django and Bitbucket's RESTful API.Features a centralized issue tracker, repository subscription manager, and reports/graphs which enumerate the commit/issue activity of each user in a subscribed repository.

Technetium's frontend is build using Twitter Bootstrap. The centralized issue tracker aggregates issue data from subscribed repositories, and has been sped up through the use of Python grequests . Showing more issues and filtering is done using AJAX for responsive design.

Smarthouse For Everyone

JavaScript • NodeJS • Arduino • iOS • C • Django • Python

A plug-an-play device for remote control of anything that is plugged into it. The motiviation behind this project is to allow home automation for the visually impaired and physically challenged.

Hardware is built using Arduino along with a Arduino Wi-fi Shield for receiving web requests. Backend server is built using Node.js and Express.js. Rooms can be configured using the iOS mobile application, and are stored through the server using Mongoose. Requests from the iOS application to the hardware is handled using the server as a backend using the node.js requests library. A web interface is also built using Django .

El-Gamal Attack

Cryptography • Python

A Python project with explores the semantic secureness of pycrypto's implementation of El Gamal. Using the Chinese Remainder Theorem, we can determine that some of the information of a transmitted message M is leaked. This information is efficiently computable by an adversary, showing that pycrypo's El Gamal implementation is not semantically secure.

Caesium

JavaScript • Node.js • Redis • Python • Socket.io

A real-time notification system built for integration into Graphyte, an ongoing research project at the G.L.A.S.S Lab at the City College of New York.

By using Node.js in combination with both its socket.io and redis client, a simple notification streaming system can be built using Redis's pubsub implementation. Whenever a new client is connected to Graphyte, they are automatically subscribed to a new channel through the Node.js Redis client, and can then recieve messages in real-time to get feedback on their homework solutions.
